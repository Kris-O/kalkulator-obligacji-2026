# kalkulator-obligacji-2026
Kalkulator obligacji indeksowanych inflacją, prezentuje rosnącą wartość obligacji z uwzględnieniem inflacji oraz regularnych płatności kuponowych. Narzędzie umożliwia szybkie porównanie różnych produktów i zmian parametrów (kwota, okres), co pozwala ocenić potencjalny zwrot w krótkim czasie.
